Here is a list of Haiku-related mailing lists. Please take a moment to review our mailing list etiquette before posting to the lists. It also explains how to unsubscribe.

In order to limit spam, you need to subscribe to a list before you can send messages to it. This allows you to receive all replies to your messages sent to the list, as well as all other discussions.

Haiku General Mailing List

This is a general discussion list. Most discussions on Haiku should happen here.

Subscribe | Message archive | RSS feed


Development Mailing Lists

Main Development List

This mailing list is for discussions related to development of Haiku itself. All Haiku maintainers should subscribe here. Please do not use this list for user support, feature requests or bug reports, only for discussing implementation details. Discussions about 3rd-party applications development should take place on haiku-3rdparty (even if you need information about Haiku internals).

Subscribe | Message archive | RSS feed

Security Mailing List

A low traffic mailing list for Haiku, Inc. to announce critical security vulnerabilities in Haiku.

Subscribe | Message archive | RSS feed

Third Party Development List

Development of third party applications that run on Haiku are discussed in this list. (e.g. new native software to run on Haiku).

Subscribe | Message archive | RSS feed

HaikuPorts List

The porting of existing software to run on Haiku is discussed in this list.

Subscribe | Message archive

Archives of the lists prior to July 1, 2015 can be found at GMANE, separated into the "users" archive and the "developers" archive.

Git Commits Log

This list logs all the Git commits to the Haiku source repository. Subscription for Haiku developers is mandatory in order to not miss important comments regarding their commits. Old Message archive (prior to 2009-10-04, hosted at berlios.de) has been lost.

Subscribe | Message archive | RSS feed

Trac bugs log

This list logs all the tickets logged in the Haiku Trac site. This is supposed to be a read-only list; if you have comments to a bug, write them in Trac instead. Old Message archive (prior to 2009-11-13, hosted at berlios.de) has been lost.

Subscribe | Message archive | RSS feed

Documentation

This list is to facilitate communication between people that work on documentation for Haiku.

HaikuDepot Web

This list is to facilitate communication between people that work on server-side support for HaikuDepot.
